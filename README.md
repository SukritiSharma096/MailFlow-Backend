# Mail Flow Backend

Mail Flow Backend is a Spring Boot–based application designed to manage and automate email workflows. It provides APIs for handling email operations such as fetching inbox messages, sending emails, forwarding, deleting, and managing attachments.

## 🚀 Features

- Email account integration (Inbox, Send, Forward, Delete)
- Attachment handling
- JWT-based authentication and authorization
- Role-based access control
- Scheduler APIs for automation
- ClickUp API integration
- Swagger API documentation

## 🛠 Tech Stack

- Java
- Spring Boot
- Spring Security (JWT)
- MySQL
- Swagger (OpenAPI)
